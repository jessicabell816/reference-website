Looking Back at CSS

Linking CSS to HTML

  • href: Used to point to the location of the CSS file (the path).
  • rel: This attribute is used to define this item as a stylesheet.
  • Example: rel="stylesheet" href="../css/style.css

Commit Messages

  • Minimum 3 words & 10 Characters
  • Proper grammer and spelling
  • Starts with capital, ends with no period
  • Must start with an imperative verb: Create, Fix, Add, Solve, etc.

Syntax of CSS

  • Rule set: Composed of a selector and one or many declarations surrounded by curly braces {}.
  • Selector (ex.h1):Identifies the HTML element being targeted for styling.
  • Declarations: Assigns a value to a property; declarations must end with a semi-colon
  • Property: The type of design you want to add. Ex. width, color, border, etc.
  • Value: Accepted value for property. Ex. 30px, blue, etc.
